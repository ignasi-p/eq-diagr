Models to Calculate Activity Coefficients

In DATABASE the equilibrium constants are given for zero ionic strength (I = 0), and it is possible to enter a value for the ionic strength when making a diagram in SPANA to calculate the activity coefficients using the values for temperature and ionic strength.

If the advanced program level is selected in the menu “Preferences / General options” in SPANA, then the user may select a model to evaluate activity coefficients. The following models may be selected:

  • Simplified HKF (Helgeson-Kirkham-Flowers) model.
    This is the default method in the normal program level of SPANA.
  • Davies equation
  • SIT (Specific Ion-interaction Theory)

All these models are based on the Debye-Hückel model.
