Temperature when making an input data file with DATABASE

In DATABASE the logK-files contain equilibrium constants at 25°C, and 1 bar, and temperature extrapolations are possible if data are available, for example ΔH°, etc.

Using the menu “Options / set Temperature” in DATABASE it is possible to select a temperature other than 25°C. When searching data in DATABASE an error message is displayed if enthalpy data is missing for any species, and an input data file for SPANA will then not be created. You must then change the temperature back to 25°C to be able to save your search results.

If no data is available for temperature extrapolations, you may enter your own data, see:

Temperature when making a diagram in SPANA

DATABASE writes the chosen temperature (and pressure) in the input data-file in a comment at the end of the first line, and this information is used when making a diagram, for example if ionic strength effects are calculated, or if Eh is displayed in diagrams.
