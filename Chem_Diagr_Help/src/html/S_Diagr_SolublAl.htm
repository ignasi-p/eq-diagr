Example of a solubility diagram for a specific solid phase.

If the the solubility of a solid, for example AlOOH (boehmite), has to be calculated, it is necessary to set the activity = 1 (log activity = 0) for this solid. Note that the most stable Al solid at 25°C is instead Al(OH)3 (gibbsite). Unless the activity of AlOOH is fixed equal to one, Al(OH)3 would precipitate in the calculations.

To set the activity = 1 for a solid you must first exchange a component for a reaction; in this case, exchange Al3+ with AlOOH(s).
