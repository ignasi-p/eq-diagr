Making an Input File for SPANA:

When you start the DATABASE program the chemical element “Hydrogen” is selected, and only protons (hydrogen ions) are selected.

  • Include electrons (by clicking on the [e−] button above the periodic table) if:
    • You want to make diagrams that have redox potential (Eh or pe) as an axis variable,
    • Your chemical system has elements that have several redox states, for example Fe(II) and Fe(III).
  • Click on any element to display its components (metal ions or ligands).
    • Organic ligands may be found when you click on Carbon.
  • Click on a component to select it. Click on a selected component to un-select it.

  • Use the menu “File / Search Reactions” to search the database with the selected components.

    On the pannel at the bottom-right (reactions found) point to a reaction product with the mouse, or select it with either the keyboard or the mouse, to display information on the status line at the bottom: reaction, equilibrium constant, and reference.

    You can remove a reaction by right-clicking with the mouse (or pressing the “Delete” key if using the keyboard).

    Show details on the reaction and its reference by right-clicking with the mouse (or pressing the “space bar” key if using the keyboard).

  • The menu “File / Save and Exit” will create an input file and then run SPANA. A database search is performed first if needed.

Working without the mouse:

  • Within the periodic table you can move with:
    • Up and Down arrow keys: move to next/previous row.
    • “Tab”; Shift-“Tab” and Left and Right arrow keys: move to next/previous element.
  • The components-lists and the reaction-list have an underlined letter in the caption texts. Type Alt-letter to jump to that list.
  • The space-bar acts as a mouse button: e.g., click on a component to selected it.
  • Menus have an underlined letter and they may be invoked by using the Alt-letter combination. For example select the menu “File” by typing Alt-F.
  • The “Tab” and Shift-“Tab” keys will move the focus to different fields in all windows. “Tab” moves the focus in one direction (usually downwards) and Shift-“Tab” moves in the opposite direction.
  • To exit a window:
    • Alt-Q and the “Esc” key are equivalent to a “Cancel” button (quit).
    • Alt-X and Alt-Enter will exit any window: they are equivalent to an “OK” button.
    • If there are “OK” and “Cancel” buttons: push the space bar on the button (change the focus between buttons with the Tab or Shift-Tab keys).
