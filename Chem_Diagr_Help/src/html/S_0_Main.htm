Chemical Equilibrium Diagrams
 
  SPANA:

Chemical equilibrium diagrams can easily be made using SPANA.

SPANA is a graphic user interface to the SED-PREDOM programs which perform the calculations needed to create chemical equilibrium diagrams.

SPANA can also call DATABASE to make diagrams based on equilibrium constants retrieved from a database.
