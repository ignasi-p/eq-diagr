Phase indicator: it gives the type of phase at the end of the name

 solid  ending in (s) for solid, (am) or (a) for amorphous, (ppt) for precipitated, (vit) for vitreous, (cr) or (c) for crystalline, . Examples: Fe2O3(cr), Fe(OH)3(am)
gas ending in (g), like CO2(g)
liquid ending in (l), like Hg(l)
aqueous any name not ending in (s), (am), (a), (ppt), (vit), (cr), (c), (g) or (l).
Examples: Na+, Fe3+, I3-.
Uncharged aqueous species: Fe(OH)3, CO2.
It is not necessary to add “(aq)” at the end of the name for an uncharged (electrically neutral) species.
