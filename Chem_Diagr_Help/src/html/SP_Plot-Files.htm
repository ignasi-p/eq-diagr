SED & PREDOM

Plot-Files

The programs SED and PREDOM read input data-files and create output plot-files. Plot-files have “plt” as the file name extension, while input data-files have “dat” as extension.

See also:

  • Plot-file format
  • Make it pretty: change the diagram

Display and Print Diagrams

You can display and print plot files with the SPANA program.

There are also two programs to convert plot-files to other formats:

  • PlotPDF: converts a plot-file to PDF.
  • PlotPS: converts a plot-file to PostScript.

These programs may be run in batch-mode. This makes it possible to automatically manage large lists of diagrams. A list of available command-line arguments may be obtained by giving the argument “-?”.

These programs have an option (command-line argument) to read the colours stored in SPANA’s “ini”-file. With this option the PDF and PostScript files will have the same colours as those displayed in the diagram windows of the SPANA program.
