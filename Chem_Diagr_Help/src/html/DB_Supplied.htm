DATABASE / Databases

Supplied database files

The default database:

The default logK-file is named “Reactions.db”, it includes stoichiometry and logK° values for reactions. The corresponding elements-file is named “Reactions.elb”, it has information on which components (reactants) are listed under a given chemical element.

This database contains more than 4500 reactions involving more than 150 components (reactants), from a variety of sources (references). A list of references is found in file “References.txt”. After searching the database in DATABASE, you may double-click on any of the reactions found to display the reference for that reaction.

Data for temperature extrapolations is available only for a few reactions.

A few organic ligands are available under the element Carbon.
