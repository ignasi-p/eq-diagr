Concentration units in diagrams

The equilibrium constants in the DATABASE program are for 25°C and 1 bar pressure. Under such conditions the density of water is close to 1 g/cm3 (see Table), and for diluted aqueous solutions molar and molal concentration units are practically the same.

In other conditions, that is, at higher temperature (or pressure), or at higher ionic strength the calculations use molal concentration units. See also temperature effects on equilibrium constants and activity coefficient models.

The programs SED & PREDOM make calculations based on the equilibrium constants, which are in molal scale, and using the activity coefficients, also for molal concentrations.

Because at 25°C the molar and molal concentration units are practically the same, the diagrams display concentrations in molar units. See this example. At temperatures above ##°C molal concentration units are displayed in the diagram. See this example.

If the user wishes to have diagrams displaying the concentrations in other units, it is possible to run the programs SED & PREDOM using special command-line arguments. When using SPANA two keys in the “ini”-file control how the concentrations are displayed:

  • Calc_concentration_notation
  • Calc_concentration_units
Alternatively, one can change the plot file itself. See: making changes in a diagram.
