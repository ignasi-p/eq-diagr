Convert diagrams to PDF

Plot-files may be converted into PDF with hte PlotPDF program:

  • The menu “Convert to ... / PDF” in a diagram window runs the PlotPDF program to convert a diagram into PDF format.
  • It is also possible to run PlotPDF in batch-mode.

With a PDF file it is possible to send a diagram to colleagues that do not have the “Chemical Equilibrium Diagrams” software installed in their computers. PDF files have names ending with “pdf” and they may be viewed and printed using other software. Some available PDF viewers for Windows:

 •
  www.adobe.com
 •
    www.foxitsoftware.com
 •
   Sumatra pdf viewer

The files produced by the “Chemical Equilibrium Diagrams” software have a page size of 20.92 × 27.94 cm (8.25 × 11 inches), i.e. the width of A4 and the height of Letter paper.
